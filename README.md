# Задание по курсу "Информационный поиск"

## Цель
Создание поисковой системы с нуля на C++ (без STL для основных структур) для корпуса из ~30 000 русскоязычных страниц.

## Источники корпуса
- https://dic.academic.ru/ — академические словари
- https://gramota.ru/ — справочная служба по русскому языку

## Компоненты
- **robot.py** — краулер на Python с записью в MongoDB
- **cleaner.py** — извлечение чистого текста
- **customcontainers.hpp** — самописные HashMap и HashSet
- **main.cpp** — токенизация, стемминг, построение булева индекса, поиск

## Запуск
1. Запустить MongoDB: `mongod`
2. Собрать корпус: `python3 robot.py config.yaml`
3. Очистить текст: `python3 cleaner.py`
4. Скомпилировать: `make`
5. Запустить поиск: `./main`
